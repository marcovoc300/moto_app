# 🏍️ MotoApp: Gestión Integral de Viajes y Mantenimiento

## 📝 1. Introducción
**MotoApp** es una solución tecnológica diseñada para la comunidad de motociclistas en Argentina. Esta base de datos centraliza la experiencia del "rider", permitiendo un control total sobre sus recorridos y el estado mecánico de su moto.

## 🎯 2. Objetivo
El objetivo principal de este proyecto es proveer una estructura relacional sólida que permita:
* **Planificar rutas:** Vincular provincias, localidades y puntos de interés.
* **Controlar el Mantenimiento:** Registrar servicios técnicos y cambios de repuestos.
* **Analizar Gastos:** Clasificar consumos (nafta, peajes, comida) para obtener métricas de costos.

## ⚠️ 3. Situación Problemática
Muchos motociclistas enfrentan una **fragmentación de la información**. Los datos de mantenimiento suelen quedar en libretas físicas y los gastos se pierden en tickets de papel. Esta falta de integración provoca incertidumbre económica y riesgos de seguridad por olvido de mantenimientos críticos.

## 💼 4. Modelo de Negocio
MotoApp opera bajo un modelo donde interactúan tres actores principales:
* **Pilotos (Usuarios):** Registran su actividad y consumen información.
* **Centros de Servicio (Mecánicos):** Proveen soporte técnico catalogado.
* **Sector Turístico (Alojamientos):** Establecimientos vinculados a las rutas.
